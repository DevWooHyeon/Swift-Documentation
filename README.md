### Language Guide

|Title|Overview|
|:---:|:-------|
|[The Basics <br> (기본)]()|Work with common kinds of data and write basic syntax. <br> (일반적인 종류의 데이터로 작업하고 기본 구문을 작성합니다.)|
|[Basics Operators <br> (기본 연산자)]()|Perform operations like assignment, arithmetic, and comparison. <br> (할당, 산술, 비교 등의 작업을 수행합니다.)|
|[String and Characters <br> (문자열과 문자)]()|Store and manipulate text. <br> (텍스트를 저장하고 조작합니다.)|
|[Collection Types <br> (콜렉션 타입)]()|Organize data using arrays, sets, and dictionaries. <br> (배열, 집합, 그리고 딕셔너리를 사용하여 데이터를 구성합니다.)|
|[Control Flow <br> (제어 흐름)]()|Structure code with branches, loops, and early exits. <br> (분기, 루프, 조기 종료로 코드를 구성합니다.)|
|[Functions <br> (함수)]()|Define and call functions, label their arguments, and use their return values. <br> (함수를 정의 및 호출하고, 인수에 레이블을 지정하고, 반환 값을 사용합니다.)|
|[Closures <br> (클로저)]()|Group code that executes together, without creating a named function. <br> (명명된 함수를 생성하지 않고 함께 실행되는 코드 그룹입니다.)|
|[Enumerations <br> (열거형)]()|Model custom types that define a list of possible values. <br> (가능한 값 리스트로 정의한 사용자 정의 타입을 모델링합니다.)|
|[Structures and Classes <br> (구조체와 클래스)]()|Model custom types that encapsulate data. <Br> (데이터를 캡슐화하는 사용자 정의 타입을 모델링합니다.)|
|[Properties <br> (프로퍼티)]()|Access stored and computed values that are part of an instance or type. <br> (인스턴스 또는 타입의 일부인 저장 및 계산된 값에 접근합니다.)|
|[Methods <br> (메서드)]()|Define and call functions that are part of an instance or type. <br> (인스턴스 또는 타입의 일부인 함수를 정의하고 호출합니다.)|
|[Subscripts <br> (서브 스크립트)]()|Access the elements of a collection. <br> (콜렉션의 요소에 접근합니다.)|
|[Inheritance <br> (상속)]()|Subclass to add or override functionality. <br> (기능을 추가 또는 재정의 하기 위한 하위 클래스 입니다.)|
|[Initialization <br> (초기화)]()|Set the initial values for a type’s stored properties and perform one-time setup. <br> (타입의 저장된 프로퍼티에 초기값을 설정하고 초기 설정을 수행합니다.)|
|[Deinitialization <br> (초기화 해제)]()|Release resources that require custom cleanup. <br> (정리가 필요한 리소스를 해제합니다.)|
|[Optional Chaining <br> (옵셔널 체이닝)]()|Access members of an optional value without unwrapping. <br> (언래핑 없이 옵셔널 값의 멤버에 접근합니다.)|
|[Error Handling <br> (에러 처리)]()|Respond to and recover from errors. <br> (에러에 응답하고 복구합니다.)|
|[Concurrency <br> (동시성)]()|Perform asynchronous operations. <br> (비동기 작업을 수행합니다.)|
|[Macros <br> (매크로)]()|Use macros to generate code at compile time. <br> (컴파일 타임에 코드를 생성하기 위해 매크로를 사용합니다.)|
|[Type Casting <br> (타입 캐스팅)]()|Determine a value’s runtime type and give it more specific type information. <br> (값의 런타임 타입을 결정하고 보다 구체적인 타입 정보를 제공합니다.)|
|[Nested Types <br> (중첩된 타입)]()|Define types inside the scope of another type. <br> (타입의 범위안에 타입을 정의합니다.)|
|[Extensions <br> (확장)]()|Add functionality to an existing type. <br> (기존 타입에 기능을 추가합니다.)|
|[Protocols <br> (프로토콜)]()|Define requirements that conforming types must implement. <br> (준수하는 타입이 구현해야 하는 요구사항을 정의합니다.)|
|[Generics <br> (제네릭)]()|Write code that works for multiple types and specify requirements for those types. <br> (여러 타입에 작동하는 코드를 작성하고 해당 타입에 대한 요구 사항을 지정합니다.)|
|[Opaque and Boxed Types <br> (불투명한 타입 및 박스 타입)]()|Hide implementation details about a value’s type. <br> (값의 타입에 대한 자세한 정보를 숨깁니다.)|
|[Automatic Reference Counting <br> (자동 참조 카운팅)]()|Model the lifetime of objects and their relationships. <br> (객체의 라이프타임과 객체 관계를 모델링합니다.)|
|[Memory Safety <br> (메모리 안전성)]()|Structure your code to avoid conflicts when accessing memory. <br> (메모리에 접근할 때 충돌을 방지하도록 코드를 구조화합니다.)|
|[Access Control <br> (접근 제어)]()|Manage the visibility of code by declaration, file, and module. <br> (선언, 파일, 모듈별로 코드의 가시성을 관리합니다.)|
|[Advaced Operators <br> (고급 연산자)]()|Define custom operators, perform bitwise operations, and use builder syntax. <br> (사용자 정의 연산자를 정의하고, 비트 단위 연산을 수행하고, 빌더 구문을 사용합니다.)|
